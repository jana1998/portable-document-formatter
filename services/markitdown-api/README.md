# MarkItDown FastAPI Service

Small FastAPI service that accepts uploaded files and returns Markdown generated by Microsoft's MarkItDown library.

## Endpoints

- `GET /health` returns a basic service health payload.
- `POST /convert` accepts a multipart `file` upload and returns Markdown plus conversion metadata.

## Run

```bash
uv sync --python 3.13 --group dev
uv run uvicorn markitdown_api.app:app --app-dir src --reload
```

The API will then be available at `http://127.0.0.1:8000`.

## Test

```bash
uv run pytest
```
